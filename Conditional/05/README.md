**Задание**  
Дано три числа, определить, являются ли они сторонами прямоугольного треугольника.  

**Входные данные**  
Три целых числа.  

**Выходные данные**  
Являются ли числа сторонами прямоугольного треугольника.  

**Пояснение**  

**Тест**  
Входные данные:  
1 2 3  
Выходные данные:  
Нет.  
Входные данные:  
3 4 5  
Выходные данные:  
Да.  
Входные данные:  
5 3 4  
Выходные данные:  
Да.
