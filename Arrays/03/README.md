**Задание**  
Найти в массиве два наибольших значения. Пример: для массива 1, 1, 2, 2, 3, 1; ответом будет 2 и 3.  

**Входные данные**  
Массив целых чисел.  

**Выходные данные**  
Два наибольших значения в массиве.  

**Пояснение**

  </tr>

  </tr>
