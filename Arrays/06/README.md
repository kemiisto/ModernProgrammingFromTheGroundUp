**Задание**  
Распечатать в разных строках журнала участки символьного массива состоящие из одинаковых символов. Пример, для массива: a, a, v, a, v, s, s, s, ответом будет:
a a;
v;
a;
v;
s s s.  

**Входные данные**  
Массив символов.  

**Выходные данные**  
Строки, состоящие их одинаковых символов.
