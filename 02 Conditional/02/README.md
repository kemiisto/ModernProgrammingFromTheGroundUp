**Задание**  
Определить, является ли число N совершенным. Совершенные числа, это числа равные сумме своих делителей. Например: 6 = 1 + 2 + 3; или 28 = 1 + 2 + 4 + 7 + 14.  

**Входные данные**  
Натуральное число N.  

**Выходные данные**  
Является ли число N совершенным.
