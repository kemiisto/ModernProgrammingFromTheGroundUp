**Задание**  
Переместить элементы числового массива по кругу. Пример, для массива: 1, 2, 3, 4, 5, ответом будет 5, 1, 2, 3, 4;  

**Входные данные**  
Массив целых чисел.  

**Выходные данные**  
Массив с перемещённым по кругу элементом.  

**Пояснение**  

**Тест**
