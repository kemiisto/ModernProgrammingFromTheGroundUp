**Задание**  
Дано два символьных массива длины N. Выяснить, совпадают ли они с точностью до символа.  

**Входные данные**  
Два символьных массива длины N.  

**Выходные данные**  
Совпадают ли массивы с точностью до символа.  

**Пояснение**  

**Тест**

  <tr>

    <td>Массивы совпадают.</td>
  </tr>
  <tr>
