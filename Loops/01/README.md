**Задание**  
Напишите три варианта (для каждой из трех форм цикла) программного фрагмента суммирования N последовательных Натуральных чисел: 1+2+3+...+N  

**Входные данные**  
Натуральное число N.  

**Выходные данные**  
1+2+3+...+N  

**Пояснение**  

**Тест**  
Входные данные:  
5  
Выходные данные:  
15 15 15
