**Задание**  
Дано два целых числа a и b. Найти значение выражения a^b, форма цикла на ваше усмотрение  

**Входные данные**  
Два целых числа a и b.  

**Выходные данные**  
a^b  

**Пояснение**  
Читатель ознакомлен только с целыми числами, поэтому b будем считать натуральным числом.
