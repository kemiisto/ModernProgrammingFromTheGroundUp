**Задание**  
Вычислить N членов ряда Фиббоначи. Ряд Фиббоначи это ряд чисел определяемый следующими условиями: a[1] = 1, a[2] = 1, a[i]= a[i-1] +a[i-2]. Выбор формы цикла на ваше усмотрение.  

**Входные данные**  
Целое число N >= 2.  

**Выходные данные**  
N членов ряда Фиббоначи.
